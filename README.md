# phpmailer-oc

Replace default mailer with [PHPMailer](https://github.com/PHPMailer/PHPMailer).

## Version
Current [PHPMailer](https://github.com/PHPMailer/PHPMailer/archive/v5.2.26.zip) version is 5.2.26.
